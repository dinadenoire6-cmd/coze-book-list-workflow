# 📖 详细配置指南

## 前置要求

在开始使用工作流前，请确保你已经完成以下准备工作：

### 1. Coze 账户设置

- ✅ 已注册 Coze 账户
- ✅ 账户已实名认证
- ✅ 账户信用分充足（建议 >= 100）
- ✅ 已配置默认的 LLM 模型（推荐 GPT-4）

### 2. 第三方平台授权

#### 抖音授权
1. 登录 [抖音开放平台](https://developer.douyin.com)
2. 创建应用（选择「短视频内容发布」）
3. 获取 `client_id` 和 `client_secret`
4. 在 Coze 中配置抖音 API
5. 授予发布权限

#### 小红书授权
1. 登录 [小红书开放平台](https://open.xiaohongshu.com)
2. 创建应用
3. 获取应用凭证
4. 在 Coze 中配置小红书 API
5. 授予发布权限

### 3. 豆瓣 API 配置

1. 访问 [豆瓣 API 文档](https://www.douban.com/api/)
2. 申请 API Key
3. 在 Coze 的「豆瓣插件」中配置 API Key

## 导入工作流

### 步骤 1：下载或复制 JSON 文件

- 选项 A：从本仓库下载 `book-list-workflow.json`
- 选项 B：复制上面的 JSON 内容

### 步骤 2：打开 Coze 导入页面

1. 登录 [Coze 官网](https://www.coze.cn/space)
2. 点击左侧「资源库」
3. 点击右上角「导入」按钮

### 步骤 3：上传或粘贴

- **上传方式**：选择「上传文件」→ 选择 JSON 文件
- **粘贴方式**：选择「粘贴代码」→ 粘贴 JSON 内容

### 步骤 4：确认导入

1. 检查工作流名称和描述
2. 确认所有节点都已正确导入
3. 点击「导入」按钮
4. 等待导入完成（通常 10-30 秒）

## 配置工作流节点

### 配置 LLM 节点

1. 打开工作流编辑页面
2. 找到「✍️ AI 生成文案」节点
3. 点击节点进入编辑
4. 配置以下参数：
   - **模型**：选择 `gpt-4` 或其他可用的 LLM
   - **API Key**：输入你的 LLM 提供商 API Key
   - **温度**：设置为 `0.7`（创意强度）
   - **最大 Token**：设置为 `500`

### 配置豆瓣插件

1. 找到「🔍 从豆瓣获取书籍信息」节点
2. 配置以下参数：
   - **API Key**：输入豆瓣 API Key
   - **搜索类型**：确保选择 `book`
   - **返回字段**：保持默认或自定义需要的字段

### 配置文本转语音

1. 找到「🎙️ AI生成配音」节点
2. 配置以下参数：
   - **TTS 引擎**：选择「腾讯云」或「百度云」
   - **声音 ID**：选择女性声线（推荐温暖磁性风格）
   - **语速**：设置为 `1.1`（稍快一点更吸引人）
   - **音量**：设置为 `100%`

### 配置视频生成

1. 找到「📹 生成完整短视频」节点
2. 配置以下参数：
   - **输出格式**：选择 `MP4`
   - **分辨率**：确保为 `1080x1920`（抖音竖屏）
   - **帧率**：保持 `30 FPS`
   - **编码**：选择 `H.264`

### 配置发布节点

#### 配置抖音发布
1. 找到「🎬 发布到抖音」节点
2. 配置 API 凭证：
   - **Client ID**：输入抖音应用 ID
   - **Client Secret**：输入抖音应用密钥
   - **Access Token**：会自动获取
3. 配置发布设置：
   - **可见性**：选择 `public`（公开）
   - **允许下载**：根据需要选择
   - **允许评论**：选择 `true`

#### 配置小红书发布
1. 找到「🔴 发布到小红书」节点
2. 配置 API 凭证：
   - **App ID**：输入小红书应用 ID
   - **App Secret**：输入小红书应用密钥
3. 配置发布设置：
   - **可见性**：选择 `public`（公开）
   - **自动配图**：根据需要选择

## 测试工作流

### 第一次运行测试

1. 打开配置好的工作流
2. 在「📖 输入书名」字段输入一个知名书籍，例如：`活着`
3. 点击「运行」或「执行」按钮
4. 观察工作流执行过程：
   - 应该看到节点逐个执行
   - 每个节点完成后显示绿色 ✓ 标记
   - 预期执行时间：1-2 分钟

### 验证输出

完成后，检查以下输出：

- ✅ **文案**：是否生成了吸引人的推荐文案
- ✅ **配音**：是否生成了清晰自然的配音
- ✅ **封面**：是否生成了专业的视频封面
- ✅ **视频**：是否生成了完整的短视频文件
- ✅ **发布**：是否成功发布到抖音和小红书

### 常见问题排查

| 问题 | 可能原因 | 解决方案 |
|------|--------|--------|
| 工作流卡在某个节点 | 该节点 API 调用超时 | 检查网络连接，重新运行 |
| 豆瓣无法找到书籍 | 书籍名称输入错误 | 确保输入的书名正确，尝试使用 ISBN |
| 配音效果差 | LLM 生成的文案不适合配音 | 修改 LLM 提示词使文案更自然流畅 |
| 无法发布到抖音 | API 授权过期或权限不足 | 重新授权，检查账户是否被限制 |
| 视频分辨率不对 | 画布配置错误 | 确保宽度 1080、高度 1920 |
