# 🆘 常见问题解答

## 导入问题

### Q1: 导入时显示「JSON 格式错误」

**原因**：JSON 文件格式不正确或有特殊字符

**解决方案**：
1. 使用在线 JSON 验证工具检查格式
2. 确保没有复制了额外的空白字符
3. 重新下载最新的 JSON 文件
4. 尝试使用文本编辑器而不是 Word 打开文件

## 运行问题

### Q2: 工作流一直在运行，不会停止

**原因**：可能某个节点处理时间较长或网络连接问题

**解决方案**：
1. 等待 2-3 分钟再试
2. 检查网络连接
3. 点击「停止」按钮强制停止
4. 检查 API 配置是否正确
5. 查看日志找出卡住的节点

### Q3: 豆瓣节点显示「无法找到书籍」

**原因**：
- 书籍名称拼写错误
- 豆瓣 API 配置不正确
- API 调用次数超限

**解决方案**：
1. 检查书籍名称是否正确
2. 尝试使用 ISBN 代替书名
3. 等待几分钟后重试（可能是 API 限流）
4. 验证豆瓣 API Key 是否有效
5. 检查 API 配额是否充足

## 发布问题

### Q4: 发布到抖音失败

**原因**：
- API 授权过期
- 账户权限不足
- 视频内容违规
- 账户被限制

**解决方案**：
1. 重新授权抖音 API
2. 检查账户是否有发布权限
3. 检查视频内容是否合规
4. 查看是否有风控提示
5. 尝试手动发布测试
